#   AI Student Impact Analysis 
 
##   Project Overview 
 
This project investigates how Artificial Intelligence (AI) influences student academic performance, study habits, and overall learning outcomes. 
 
The analysis combines Excel, SQL, Python, and Power BI to transform raw student data into meaningful insights regarding GPA improvement, AI usage patterns, burnout risk, academic disciplines, and prompt engineering skills. 
 
The goal is to demonstrate an end-to-end data analytics workflow, from data cleaning and exploration to visualization and dashboard creation. 
 
 
 
##   Business Questions 
 
This project aims to answer the following questions: 
 
1.	Does AI usage improve student GPA? 
2.	Which academic majors use AI the most? 
3.	Is heavy AI usage associated with burnout risk? 
4.	How does AI usage vary across academic years? 
5.	Does prompt engineering skill influence academic performance?

##   Key Findings 
 
### GPA Improvement 
 
Students with moderate AI usage achieved the highest average GPA improvement. ### Major Analysis 
 
STEM students demonstrated the highest average AI usage. 
 
### Burnout Risk 
 
Students classified as high burnout risk reported greater AI usage than other groups. 
 
### Academic Year Analysis 
 
AI usage patterns differed across academic years. 
 
### Prompt Engineering Skills 
 
Prompt engineering skill level showed only minor differences in GPA improvement.
